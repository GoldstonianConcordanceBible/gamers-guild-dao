# Roadmap

The GGUILD roadmap is designed to move from concept to functioning ecosystem through clear phases.

The purpose of the roadmap is not to create hype. The purpose is to show how the project becomes more real over time.

## Phase 1 — Documentation and Narrative

This phase focuses on building the explanation layer of the project.

Key priorities include:

- GitBook structure
- GitHub documentation
- token language
- governance language
- public-facing clarity
- alignment between GitBook, GitHub, and Pump.fun

## Phase 2 — Community Activation

This phase focuses on building the early community.

Key priorities include:

- onboarding first community members
- creating Discord and Telegram flow
- directing people into the docs
- establishing message discipline
- making the project understandable to new users

## Phase 3 — Reputation and Rewards

This phase introduces stronger participation logic.

Key priorities include:

- contributor recognition
- community reward experiments
- reputation scoring concepts
- early contributor pathways
- stronger alignment between effort and recognition

## Phase 4 — Gaming Integration

This phase connects the token more directly to gaming use cases.

Key priorities include:

- creator campaigns
- tournament support models
- community engagement systems
- role-based participation
- gaming-adjacent ecosystem experiments

## Phase 5 — Governance Expansion

This phase builds more visible decision structures.

Key priorities include:

- proposal templates
- discussion process
- voting pathways
- community input loops
- governance documentation

## Phase 6 — AI Operations

This phase expands the system layer.

Key priorities include:

- analytics support
- reporting automation
- contributor scoring support
- ecosystem monitoring
- operational intelligence

## Phase 7 — SydTek University Integration

This phase deepens the academic case study value.

Key priorities include:

- course integration
- case study assignments
- documentation reuse
- esports program alignment
- long-term academic infrastructure value

## Phase 8 — Scale and Refinement

This phase focuses on long-term durability.

Key priorities include:

- improving clarity
- refining governance
- strengthening participation systems
- improving reporting
- expanding carefully and responsibly

## Final Point

The roadmap exists to turn vision into sequence.

A project becomes stronger when people can see what stage it is in and what comes next.
