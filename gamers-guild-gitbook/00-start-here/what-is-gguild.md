# What is GGUILD

GGUILD is a token built for gaming communities.

It allows gamers, creators, and contributors to:

- earn rewards  
- build reputation  
- participate in governance  
- engage in a shared ecosystem  

---

## 🧠 Simple Explanation

GGUILD is a system where:

- you contribute → you earn  
- you participate → you gain influence  
- you help grow the community → you are recognized  

---

## 🎮 What It’s Used For

GGUILD can be used for:

- gaming rewards  
- creator incentives  
- community participation  
- governance voting  
- access to features and events  

---

## 🎓 Why It’s Different

GGUILD is part of SydTek University.

That means:

- it is documented  
- it is studied  
- it is structured  
- it is built as a real system
