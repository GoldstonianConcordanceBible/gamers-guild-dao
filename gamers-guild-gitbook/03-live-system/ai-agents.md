# AI Agents

AI agents help GGUILD operate more efficiently.

They do not replace the community. They support it.

## What AI Agents Do

AI agents can:

- analyze activity
- track engagement
- assist with moderation
- update reputation scores
- generate reports

## Example Roles

### Scout Agent
Finds trends and opportunities.

### Analyst Agent
Measures performance and activity.

### Reputation Agent
Updates contributor scores.

### Treasury Agent
Supports planning and allocation insights.

## Why This Matters

Without automation, systems become slow and inconsistent.

AI agents help GGUILD scale while staying organized.
