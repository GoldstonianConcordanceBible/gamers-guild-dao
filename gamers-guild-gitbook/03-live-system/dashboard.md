# Dashboard

The dashboard is the visual layer of the GGUILD system.

It shows what is happening in the ecosystem.

## What the Dashboard Displays

- user activity
- token distribution
- top contributors
- governance activity
- ecosystem metrics

## Why It Matters

The dashboard provides:

- transparency
- visibility
- accountability

## Goal

The goal is to make the system understandable at a glance.

If users cannot see what is happening, they cannot trust it.
