# API

The API connects different parts of the GGUILD system.

## What an API Does

An API allows:

- the frontend to talk to the backend
- the dashboard to pull data
- agents to send and receive updates

## Simple Explanation

Think of the API as the messenger between systems.

## Example Uses

- fetching user data
- updating scores
- retrieving metrics
- powering dashboards

## Why It Matters

Without an API, systems cannot communicate effectively.

The API makes GGUILD a connected system instead of isolated parts.
