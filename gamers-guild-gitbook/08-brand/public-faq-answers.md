# Public FAQ Answers

Use these answers when people ask simple public-facing questions.

## What is GGUILD

GGUILD is a gaming-native utility and governance token built to support gaming communities, contributor rewards, creator incentives, and community participation.

## What makes this different

GGUILD is being built as a documented SydTek University Sub-DAO case study. That means it is structured as a real system with GitBook docs, GitHub infrastructure, roadmap visibility, and academic integration.

## What is the token for

The token is designed for participation, governance, contributor rewards, creator incentives, access, and community coordination.

## Is this just a meme token

No. GGUILD is positioned as a utility-oriented and governance-linked gaming ecosystem token with a documented long-term vision.

## How does SydTek University fit in

GGUILD serves as the live digital economy case study layer for SydTek University’s B.S. in Esports, Gaming, and AI Performance.

## What role does AI play

AI agents are intended to support analytics, reporting, contributor scoring, and ecosystem coordination.

## Where do I learn more

Start with the GitBook and GitHub repository. The documentation explains the purpose, structure, roadmap, and system design.

## What should people understand first

People should first understand that GGUILD is being built around structure, utility, participation, and documentation rather than hype.
