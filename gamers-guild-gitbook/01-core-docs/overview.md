# Overview

Gamers Guild Token (GGUILD) is a gaming-native utility and governance token.

It is designed to support:

- gaming communities
- creator incentives
- contributor rewards
- governance participation
- AI-supported ecosystem coordination

## Core Idea

GGUILD connects:

gaming + community + governance + AI + education

## Why It Exists

Gaming communities create value.

GGUILD is designed to help communities:

- coordinate
- reward contribution
- build shared systems
- operate transparently

## What Makes It Different

GGUILD is:

- documented (GitBook)
- structured (GitHub)
- connected to SydTek University
- built as a case study
- focused on utility, not hype
