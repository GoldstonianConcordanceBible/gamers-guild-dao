# Governance

Governance allows the community to participate in decisions.

## Core Idea

GGUILD governance is designed to:

- collect ideas
- structure proposals
- allow community input
- support decision-making

## What Governance Can Cover

Governance may include:

- ecosystem direction
- feature ideas
- community initiatives
- contributor recognition systems

## Principle

Governance should be:

- understandable
- structured
- visible
- documented

## Goal

The goal is to move from:

centralized decisions → community-involved decisions
