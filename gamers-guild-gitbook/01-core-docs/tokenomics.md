# Token Utility

GGUILD is designed around participation and contribution.

## Core Utilities

The token may be used for:

- contributor rewards
- community participation
- governance voting
- access to features or events
- ecosystem incentives

## Contribution Model

Users contribute through:

- gaming activity
- content creation
- community support
- system building

Contribution → recognition → rewards

## Design Principle

Utility should be:

- clear
- simple
- tied to real actions
- aligned with community growth
